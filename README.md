# Python Packager Pro 🚀

![Python Version](https://img.shields.io/badge/python-3.8+-blue.svg)
![PyQt6](https://img.shields.io/badge/GUI-PyQt6-green.svg)
![License](https://img.shields.io/badge/license-MIT-important.svg)

**Python Packager Pro** 是一款基于 **PyQt6** 开发的图形界面工具，旨在简化 Python 脚本的打包与发布流程。它将复杂的命令行操作转化为直观的 GUI 交互，帮助开发者高效完成从脚本到可执行文件的转化。

---

## ✨ 主要功能

* **多引擎支持**：集成 **Nuitka** 和 **PyInstaller** 两大主流打包工具。
* **智能环境管理**：自动检测系统 Python 环境，支持手动配置虚拟环境路径。
* **一键依赖处理**：自动扫描项目依赖，支持缺失库的一键安装。
* **高级编译选项**：
    * 支持 **MSVC** 和 **MinGW** 后端编译器自动匹配。
    * 支持多核并行加速，大幅提升 Nuitka 打包速度。
* **资源与压缩**：
    * 内置图标制作工具（基于 Pillow），支持多种尺寸转换。
    * 集成 **UPX** 及内置压缩方案，有效减小体积。
* **全流程监控**：实时输出打包日志，提供进度条反馈，并支持随时取消操作。
* **自动化清理**：打包完成后自动清理临时构建文件，保持目录整洁。

---

## 🛠️ 依赖环境

使用本项目前，请确保您的环境满足以下要求：

* **Python 3.8+** (强烈建议使用虚拟环境)
* **PyQt6**: UI 框架支持
* **Pillow**: 用于图标处理和转换
* **编译器**: 如使用 Nuitka，需安装对应的 C++ 编译器 (MSVC 或 MinGW)

---

## 🚀 快速上手

1.  **安装核心依赖**:
    ```bash
    pip install PyQt6 Pillow
    ```
2.  **启动程序**:
    ```bash
    python main.py
    ```
3.  **开始打包**:
    * **选择入口**: 导入你的 Python 主程序文件。
    * **配置**: 设定输出目录、应用图标及打包工具（Nuitka/PyInstaller）。
    * **执行**: 点击“立即打包”，程序将自动处理依赖并执行构建。
    * **完成**: 打包成功后，程序将自动打开输出文件夹。

---

## 💡 其他说明

* **多核加速**: 程序会自动检测 CPU 核心数，配置 Nuitka 以并行模式运行。
* **依赖管理**: 在打包前，程序会通过弹窗形式提醒用户确认并安装项目缺失的依赖库。
* **安全性**: 提供取消按钮，防止误操作或因配置错误导致长时间死锁。
